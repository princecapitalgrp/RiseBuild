The Six-Layer Personalization Loop

1
Raw Input checkin:{date}

Voice transcript, slider values, the named weight. Stored locally as collected. Never sent to AI.

2
Interpreted State context:{date}

Structured AI-ready context derived from Layer 1. "High cognitive load. Pre-loaded performance anxiety." This — not Layer 1 — is what reaches the server.

3
Generated Routine protocol:{date}

Morning protocol produced by AI from Layer 2 + memory patterns. Stored with prompt version for calibration tracking.

4
Execution Data (new in Block 1) execution:{date}

What actually happened. Step completions, timing deviations. Closes the loop between what was prescribed and what was executed. Did not exist in the web prototype.

5
Reflection reflection:{date}

Post-routine feedback. feltAccurate, followedIt, helpedMost, gotInWay. Raw notes stay on device.

6
Long-term Memory memory

Patterns distilled from Layers 4+5 over time. What reaches AI = pattern summary. Raw Layer 1 never referenced. This is what makes future protocols better.
